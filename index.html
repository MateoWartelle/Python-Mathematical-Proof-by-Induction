Prove by Mathmatical Induction\[\sum_{i=1}^n i = \frac{n(n+1)}{2} \]Base case n = 1\[\sum_{i=1}^1 1 = \frac{1(1+1)}{2} \]\[1=\frac{1(1+1)}{2}\]
Assume for some w = n\[\sum_{i=1}^w i = \frac{w(w+1)}{2} \]Show true for w + 1\[\sum_{i=1}^{w + 1} i = \frac{(w + 1)((w + 1)+1)}{2} \]
